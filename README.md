# 🛒 Rosaline Web — E-commerce Frontend

A modern e-commerce frontend built with React and Vite.  
Focus on clean UI, product browsing, and shopping cart experience.

---

## 👩‍💻 About the Project

This project is a frontend e-commerce application built to practice real-world React development.  
It includes product listing, filtering, cart management, and responsive design.

---

## ✨ Features

### 🛍 Products
- Fetch products from API
- Responsive product grid
- Product cards with image, price, and rating
- Category filtering
- Search functionality

### 🛒 Cart
- Add / remove products
- Increase / decrease quantity
- Real-time total price
- Persistent cart (localStorage)

### 🎨 UI
- Responsive design (mobile + desktop)
- Clean modern layout
- Simple and fast user experience

---

## 🧱 Tech Stack

- React (Vite)
- JavaScript (ES6+)
- Tailwind CSS
- Redux Toolkit (optional state management)
- DummyJSON API
